### Clone

- Clone this repo to your local machine using `https://github.com/Spencedesu/ServiceStaffStrikeBack.git`

### Setup

- Be sure to direct you terminal to the root of the repo

> Install npm packages and start 

```shell
$ npm install
$ npm install axios
$ npm start
```
